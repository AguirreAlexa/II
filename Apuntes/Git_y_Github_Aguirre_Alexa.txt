Consignas:

1. ¿Quién inventó el sistema de control de versión Git y por qué?

    Git es un sistema de control de versiones distribuido que, como ya hemos introducido, 
    fue creado por Linus Torvalds en 2005 para ser utilizado en el desarrollo del kernel de Linux, 
    un proyecto de software de código abierto con un alcance bastante amplio.

2. ¿A quién pertenece actualmente Github y por qué?

    Actualmente, GitHub pertenece a Microsoft y es que desde el nombramiento de Satya Nadella como CEO,
    la compañía ha adoptado una estrategia nueva, mucho más abierta al mundo del software libre y a la 
    integración con otras plataformas y servicios. 
    Microsoft tiene que competir con el líder indiscutible a día de hoy, la todopoderosa Amazon con AWS, 
    y busca estar en contacto directo con los desarrolladores que trabajan en esos servicios, ver sus 
    necesidades y ofrecerles herramientas que faciliten su integración con Azure.

3. ¿Hay otra forma que no sea la terminal para trabajar con Github?

    Se puede trabajar directamente desde la pagina o aplicacion de escritorio.
